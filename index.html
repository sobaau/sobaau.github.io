Genetic Algorithm - Python

This project uses a Genetic Algorithm to attempt to solve a pre-provided Zebra Puzzle The target is to get 15 of the hints correct from the Zebra Puzzle and hence find the answer to the question:
Which car was going to Port Macquarie? Which car was hired by a Canadian couple?
This is done by using constraints from the hints and using True or False (1 and 0) to calculate the total right hints and then fitness of that table on a scale of 0 to 1.

To avoid duplicates in the answer the AI is punished within the fitness function if any duplicates are found. Mating pool selection is performed with a Tournament Selection of 1000 rounds and 40 fighters, The 40 fighters selected can not be in the current mating pool and will be reselected if they are. The single strongest of these fighters are then added to the mating pool which will have a total size of 1000. After this the Parents are picked at random from the mating pool, These parents can not be the same parent. The Child is created from these Parents by selecting random parts of their tables to crossover and then once this is finished the child is mutated in random locations depending on the mutation rate provided. To avoid any Premature Convergence and hence stalling of the Algorithm it is restarted after 100 iterations.
